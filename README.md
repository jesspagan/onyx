# onyx

Nx workspace for building the Onyx design system.

## Getting started

1. Install dependencies:

	npm install

2. Verify Nx is available:

	npx nx --version

## Generate projects

Create a library:

```bash
npx nx g @nx/js:lib ui-core
```

Create a Node app:

```bash
npx nx g @nx/node:app docs-api
```

## Useful commands

```bash
npm run graph
npm run build
npm run test
npm run lint
```
